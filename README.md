MPM 物质点法实时物理模拟
📖 项目介绍
本项目基于 C++20 标准与 OpenGL 图形接口开发，实现了MPM（物质点法）实时物理模拟系统。支持任意自定义三维模型与MPM粒子系统的实时物理交互，可高效模拟流体、软体、弹性材质等多种物理效果，具备实时渲染、动态交互、高适配性的特点。
项目已内置全部依赖库，无需额外配置第三方环境，开箱即可编译运行，适合物理仿真学习、可视化模拟与二次开发。
✨ 核心特性
- 前沿技术栈：采用 C++20 最新语法特性开发，代码简洁高效、性能优异，兼容现代编译环境
- 实时物理模拟：基于标准 MPM 物质点算法，实现高精度粒子物理动力学计算
- 自由模型交互：支持任意自定义三维模型导入，与粒子系统实时碰撞、受力交互
- OpenGL 实时渲染：硬件加速图形渲染，模拟效果流畅、可视化效果清晰
- 零依赖配置：仓库内置所有所需依赖库，无需手动下载、配置第三方库
💻 环境要求
- 编译器：支持 C++20 标准
- 构建工具：CMake 3.20+
- 系统：Windows 跨平台支持
🔨 构建编译
本项目支持纯 CMake 一键构建，无需配置任何额外 CMake 参数，所有依赖、编译规则已内置配置文件中。
通用编译步骤
1. 克隆项目到本地
        git clone https://github.com/LittleGreenHand/OpenGL-MPM.git
cd 项目根目录
2. 创建编译目录并构建
        # 创建build编译文件夹
mkdir build && cd build

# CMake 自动配置（无额外参数）
cmake ..

# 编译项目
make -j8  # Linux/macOS
# 或 Visual Studio 编译（Windows）
cmake --build .
3. 运行生成的可执行程序即可启动模拟
